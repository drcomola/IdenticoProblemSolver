# Template caso minimo

Usa questi nomi file per un caso con 3 viste pre e 3 viste post:

```text
before-upper-arch.jpg
after-upper-arch.jpg
before-lower-arch.jpg
after-lower-arch.jpg
before-front.jpg
after-front.jpg
```

Poi copia la cartella dentro la categoria corretta e rinominala con lo slug del caso.
